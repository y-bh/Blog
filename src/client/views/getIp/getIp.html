使用说明

* 生成API链接，调用GET请求即可返回所需的IP结果

* 您的所有访问操作均有日志保存，望遵纪守法

* 请使用合理的频率去调用API接口，暴力调用将导致您的客户端IP被防火墙封禁

提取接口
  • 属性
  • 内容
  • http://api.tianqiip.com/getip
  • GET
  • utf-8
  • code: 错误编号
    Msg: 描述信息

  • 返回内容的标签或字段类型可在产品管理里通过设置提取格式来定制；
    类型为txt文本的定制标签如下：
    ip：port自定义分隔符；
    类型为json格式的响应字段如下：
    data：[
    {
    ip：IP
    port：端口
    expire：有效时间
    city：城市
    isp：运营商
    }
    ]

  • 1000: 获取成功
